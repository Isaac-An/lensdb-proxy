# **App Name**: Appleye Lens Database

## Core Features:

- Excel Import: Import product data from Excel files.
- Data Synchronization: Automatically sync the imported Excel databases on a daily basis.
- Product Search: Search for products using keywords.
- Advanced Filtering: Apply filters based on product attributes (Sensor Size, EFL, Max Image Circle, F. No., FOV - Diagonal, FOV - Horizontal, FOV - Vertical, TTL, TV Distortion, Relative Illumination, Chief Ray Angle, Mount Type, Lens Structure).
- AI Product Insights: Use an AI tool to generate human-readable summaries of trends in product data.
- Product Details View: View detailed product information (specifications, pricing, attributes).
- Excel Upload: Upload newer lenses via Excel files.

## Style Guidelines:

- Primary color: Based on the appleye.co.il website color palette, use a dominant color that reflects technological precision and clarity.
- Background color: A light, neutral color, sampled from appleye.co.il, to provide a clean and non-distracting backdrop.
- Accent color: A vibrant color from appleye.co.il, used strategically to highlight important actions and insights.
- Font: 'Montserrat' for all text elements to maintain a consistent and modern look, based on the style of appleye.co.il.
- Modern and clear icons for navigation and product attributes, consistent with the visual language of appleye.co.il.
- Clean and structured layout with clear sections for data import, search, filters, and product details.
- Subtle transitions and animations for loading data and updating product information.
